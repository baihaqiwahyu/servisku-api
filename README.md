# Servisku API - Backend

Backend Servisku dibuat menggunakan Laravel dan MySQL. API ini digunakan untuk menyediakan data motor dan komponen servis yang ditampilkan pada frontend React.

## Fitur

- API daftar motor
- API detail motor berdasarkan slug
- Relasi motor dengan komponen servis
- Seeder data motor dan komponen servis
- Database MySQL
- Integrasi dengan frontend React

## Teknologi

- Laravel
- PHP
- MySQL / MariaDB
- XAMPP
- Eloquent ORM
